# Backend Updates

We have enhanced several of our backend components.

## Enhancements

- **Replace Konga with Kong Manager** - We had previously upgraded to Kong 
  Gateway 3.x which includes support for Kong Manager. We have replaced 
  Konga with Kong Manager to improve maintainability, security, and 
  compatibilty with future Kong releases.
- **Rate Limiting** - We have implemented rate limiting to protect
  our backend services from abuse, reduce server load, ensure 
  system stability, and provide the team additional visibility into
  system usage.
- **Thanos Configuration** - We have adjusted our Thanos configuration
  to reduce syncing frequency and removing unneeded flags.
