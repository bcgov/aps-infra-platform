# API Services Portal Release 3.0.6

The API Services Portal has been updated with two fixes.

## Fixes

- **Sanitized Directory API response** - Removed the `services` payload from
  Directory API `directory/{id}` responses.
- **Updated two-tiered-hidden icon** - Two-tiered-hidden products now display a
  lock icon (instead of the previous earth/public icon), providing a more
  accurate visual representation of restricted access.
